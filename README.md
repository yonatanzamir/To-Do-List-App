# Todo-Application 

![node.js]

### Built With
* React
* Node.js



## Usage

* Clone the repository, install node packages and verify routes locally

```bash
git clone https://github.com/yonatanzamir/To-Do-List-App/
cd To-Do-List-App/api
npm install
npm start
cd To-Do-List-App/Client
npm install
npm start
```
